# RPIL 2526 - Recherche de Mentor

Application de matching mentor-mentoré - Projet intégrateur IFRI.

## 🛠️ Technologies

- **Frontend** : HTML / CSS / JavaScript
- **Backend** : Python / Flask
- **Base de données** : PostgreSQL
- **Déploiement** : Railway

## 🚀 Lien du projet

https://rpil2526athindehoufrejus-production.up.railway.app/

## 📁 Structure

- `app.py` : Serveur Flask + API de matching
- `models.py` : Modèle de base de données
- `seed.py` : Remplir la base avec 3 mentors
- `static/index.html` : Interface utilisateur

## 🧠 Algorithme de matching

- **20 points** par compétence commune
- **+10 points** si la filière correspond
- Tolérance horaire de **±1 heure**
